\# Pneumonia Detection using Deep Learning



\## Description

This project detects pneumonia from chest X-ray images using a pretrained ResNet18 model built with PyTorch.



\## Features



\- Image Upload

\- Pneumonia Prediction

\- Confidence Score

\- Probability Bars

\- Streamlit Web App



\## Technologies



\- Python

\- PyTorch

\- Streamlit

\- OpenCV

\- NumPy

\- Matplotlib



\## Model Performance



Accuracy : 88.62%



ROC-AUC : 95.42%



\## Run



streamlit run app.py
